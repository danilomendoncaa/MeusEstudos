Principais Formatações

Negrito/Destaque

Nesta frase temos um termos em negrito usando a Tag B (não semântica).

Nesta frase temos um termo em destaque usando a Tag STRONG (semântica).

Itálico/Ênfase

Nesta frase temos um termo em itálico usando a Tag I (não semântica).

Nesta frase temos um termo em Ênfase usando a tag EM (semântica).

Texto Marcado

Podemos também usar um texto marcado usando a Tag MARK

Texto Deletado

Podemos marcar um texto como Deletado para indicar que ele deve ser lido, mas não considerado, para isso usamos a tag del

Texto Inserido

Podemos marcar um texto como Inserido para dar uma ênfase e indicar que ele foi adicionado depois, para isso usamos a tag ins

Texto Sobrescrito

Para inserir coisas do tipo x2+3, para isso usamos a Tag SUP

Texto Subscrito

Para inserir coisas do tipo H2O, para isso usamos a Tag SUB
